# Agent Rules

1. Work only on `AllVision`.
2. Follow TDD for every cycle.
3. Run full repo tests before completion.
4. Produce/update `docs/specs/<task-key>.md` for meaningful tasks.
5. Include project-specific UX research evidence in docs.
6. End each run with `commit + push`.
7. Stop automatically on `project_completed` or `awaiting_user_input`.
